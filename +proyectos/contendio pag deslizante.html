O QUE SÃO AS TAGS?

As tags são utilizadas para marcar o início de um elemento HTML e estão normalmente entre sinal de menor < e maior>. Um exemplo de uma tag é: <h1>.

A maioria das tags devem ser abertas <h1> e fechadas </h1> para funcionarem.

O QUE SÃO OS ATRIBUTOS?

Os atributos contêm informações adicionais. Os atributos assumem a forma de uma tag de abertura e informações adicionais são colocadas dentro.

BASE DE UMA PÁGINA HTML

<!DOCTYPE html> - Esta tag específica a linguagem que você vai escrever na página. Neste caso, a linguagem é HTML5 e que a partir deste momento vamos escrever em código HTML. <head> - Aqui é onde vão todos as meta tags da página - coisas destinadas principalmente a motores de busca e outros programas de computador. <body> - Aqui é para onde vai o conteúdo da página.

O QUE É CSS?

CSS significa Cascading style Sheet ou Folhas de Estilo em Cascata. CSS descreve como os elementos HTML devem ser mostrados na tela, ou em outro documento. CSS reduz muito o tempo de trabalho, pois ele pode controlar o layout de várias páginas ao mesmo tempo. Estilos externos(não inline) devem ser salvos dentro de um arquivo .css

SINTAXE DO CSS

Seletor - O seletor é um trecho de código usado para identificar qual o elemento que irá ser modificado; Propriedade - É o aspecto que será modificado do elemento selecionado, pode ser: cor(color), cor de fundo(background-color), tamanho do texto(font-size) Valor - É o valor da propriedade. No exemplo para a propriedade cor(Color) utilizamos vermelho(red); Declaração - É a combinação da propriedade e do valor.

USO DO CSS EM UM WEBSITE

Podemos utilizar o CSS dentro de um website de três maneiras diferentes são elas: -CSS Interno -CSS Externo -CSS Inline
